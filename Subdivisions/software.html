Software Subdivision


The software stack has been divided into the cognition, firmware and driver layer and is extensively built on ROS(Robot Operating System),developed at Willow Garage which operates to integrate the different layers and different independent subsystems within each layer.

Cognition layer involves the mission planner, task state machine and the vision subsystem which enable autonomous performance of tasks by integration of functions of various lower level subsystems , and navigatsor and controller which control the motion of the vehicle. Firmware layer consists of the software which operates on the beaglebone to control various hardware components of the vehicle and enabling its communication with the Single Board Computer via socket programming.

Driver Layer functions to collect data from a multitude of sensors used in the vehicle and effectively communicate them to the cognition layer.
